# FLOW AUTOMATION - IDENTITY SYSTEM

## 🎨 Color Palette
- **Primary Background:** `#050505` (Deep Dark - Not Pure Black)
- **Brand Accent:** `#790028` (Flow Wine) -> Use for active states, glows, cursors.
- **Text Primary:** `#EDEDED` (Off-white)
- **Text Secondary:** `#888888` (Tech Gray)

## 🔤 Typography
- **Headlines (Display):** `Reachfly` -> Use ONLY for massive, short keywords (e.g., "FLOW", "DATA").
- **Body/UI:** `Gilroy-Light` -> Use for all paragraphs, navigation, and data tables.

## ⚡ UX Behavior
- **Initial Load:** Must look like a system boot sequence or data stream connection.
- **Micro-interactions:** All clickable elements must react physically (scale, glow) on hover.
